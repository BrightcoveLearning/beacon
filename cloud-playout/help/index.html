Add new channel
  1. Click to open the Create Channel dialog.
  2. Enter a name (English only, no spaces; these special characters are allowed: _ . : / = + -). Required
  3. Enter a description of the channel. Required
  4. Click on the Start Date/Time field to select a start date and time. Required
  5. If you toggle Loop Playout Content on, you may also enter an optional End Date/Time

  6. Toggling this on enables SSAI - it will also removed the Live Input option.

  7. Fill in the fields and click .
