Managing users

View, add or delete users for this account.

Add a user

  1. Enter values for a new user:
    • Email
    • First Name
    • Last Name
  2. Select a user Role.
    • Standard - User does NOT have access to Admin sections
    • Administrator - User has access to Admin sections
  3. Module Access - Select the module(s) that you want the user to have access to
  4. Click Add User.

Edit a user

  1. Click on the user to edit.
  2. Edit user details.
  3. Click Save.

Delete a user

  1. Click the delete icon associated with the user.
  2. Click OK to confirm.
