Upload Settings

These settings are associated with uploading a video in Media.

Default to Inactive - When checked, sets all uploaded videos to be inactive.

Required Fields

  • Required for activation - Include required field values for video activation.
  • Required for upload - Include required field values for video upload using the Upload page.

Upload Additional Assets - Allow poster and thumbnail images to be uploaded when uploading videos using the Upload page.

Poster Image Default Size (px) - The default poster image size used when not specified in the ingest profile.

Thumbnail Image Default Size (px) - The default thumbnail image size used when not specified in the ingest profile.

Upload Queueing

  • Allow concurrent uploads - Multiple files are uploaded concurrently.
  • Upload one file at a time - Files are uploaded one at a time. Recommended for large files.

After changing settings, click Save.
