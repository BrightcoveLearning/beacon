Live Settings

Manage fill slates and credentials for Brightcove Live streams. SSAI ad configurations are managed in Server-Side Ad Settings.

Manage Fill Slate

Fill slates are content to be served when there are gaps in a live stream ad break.

  1. Click Add Fill Slate.
  2. Enter Name - a descriptive name for your fill slate.
  3. Enter Source URL - full address of the fill slate.
  4. Click Save.
Manage Credentials

Live credentials are required when using either the Live module or the Live API.

  1. For Live Video Cloud credentials, start with API Authentication credentials .
  2. Enter Name - a descriptive name for your fill slate.
  3. Enter Source URL - full address of the fill slate.
  4. Click Save.
