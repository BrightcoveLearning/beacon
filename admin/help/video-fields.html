Video Fields Help

Video fields are metadata associated with each video in your account. You have access to a set of standard fields, and you can create your own custom fields.

Need more fields? - Each account is given a set number of custom fields. If you need more, contact customer support .

Custom fields

User-created fields.

  • Field - Toggle between show internal names and show display names for custom and standard field lists.
  • Click on a custom field name to edit.
  • Display Name - Custom field name. Double-byte characters and spaces permitted.
  • Internal Name - API identifier. Lower case, alphanumeric characters with no spaces.
  • Type - Select one of the following:
    • Text - Text input field. Maximum 1024 single-byte characters or 512 double-byte characters.
    • List - Dropdown list of values. Maximum 150 values with each value maximum of 100 characters.
  • Description - Provide user instructions for setting the field value. This appears as a hint for each video detail in the Media page.
  • Make this Field Required - The user is required to enter a custom field value before the video can be activated.
