Server-Side Ad Settings

Brightcove's Server-Side Ad Insertion (SSAI) embeds ads in your video stream to provide a TV-like viewing experience with no ad buffering.

Navigation

  • Delete - Select one or more ad configurations, and click Delete to remove them from your account.
  • Duplicate - Select an ad configuration, and click Duplicate to create an ad configuration copy. You can cancel, or edit and save the new configuration.

Ad Configuration List

  • Name - The name of the ad configuration. Click Name to view the details for the ad configuration.
  • Type - Either VOD or Live
  • Ad Server Response - The response format from your ad server.
  • Created - The creation date for the ad configuration.

Ad Configuration Details

  • Name - The name of the ad configuration. Click Name to view the details for the ad configuration.
  • Type - Either VOD or Live
  • Ad Server Response - The response format from your ad server.
  • Created - The creation date for the ad configuration.
