Server-Side Ad Settings

Brightcove's Server-Side Ad Insertion (SSAI) embeds ads in your video stream to provide a TV-like viewing experience with no ad buffering.

Navigation

  • Delete - Select one or more ad configurations, and click Delete to remove them from your account.
  • Duplicate - Select an ad configuration, and click Duplicate to create an ad configuration copy. You can cancel, or edit and save the new configuration.
  • Create Ad Configuration - Click to expand the dropdown menu, and choose one:
    • VOD - Video on Demand
    • Live - Live stream

Ad Configuration List

  • Name - The name of the ad configuration. Click Name to view the details for the ad configuration.
  • Type - Either VOD or Live
  • Ad Server Response - The response format from your ad server.
  • Created - The creation date for the ad configuration.

Ad Configuration Details

  • Type - Either VOD or Live
  • Name - The name of the ad configuration.
  • Ad Server Response - The response format from your ad server.
    • DFP Ad Rules - Proprietary format for DFP/Google for Publishers
    • DFP VMAP - Video Multiple Ad Playlist
    • Smart XML - Proprietary format for FreeWheel
    • VAST 3.0 - Digital Video Ad Serving Template
  • Ad Tag - The server URL for your ad tag.
  • Disable server-side beacons - These beacons emit events from the server to measure ad impressions and report ad errors. If disabled, events will not be fired from the server, but will be included in the ad response.
  • Save - Click Save to create a new ad configuration or save changes to an existing configuration.
