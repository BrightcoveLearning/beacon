Media Sharing

Media sharing allows publishers to upload videos once and then share the videos with other accounts..

Terminology

  • Master - The account where the video was uploaded. The master owns the content and is responsible for setting up, managing and providing content to Affiliates.
  • Affiliate - The account receiving the shared video. The affiliate can approve or reject sharing relationships.

Manage Affiliates

  • Add Affiliate - Enter the Account ID for which you want to share media with, and click Add Affiliate.
  • Relationship Status - When the affiliate account accepts the sharing relationship, the status will be Accepted.
  • Remove - Click the delete icon (trash can) associated with the Account ID that you want to remove from a sharing relationship.

Sharing Rules for All Affiliates

  • Enforce custom field matching - Requires affiliate accounts to have the same custom fields as the master account or any shares will fail. To enable, check this box and click Update Rules.

Manage Masters

  • After a Primary Language is saved, additional metadata languages can be added. This allows you to add multi-lingual metadata on the Media page.
  • Click Add Language or click on a language in the list to edit.
  • In the Language input field, Start typing your language to see values.
  • Select a language.
  • Click Save Language.
