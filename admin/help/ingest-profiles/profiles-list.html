Ingest Profiles List

The ingest profile defines the number and type of renditions created for each video upload.

Show Display Name / Show API Name - Toggle between profile names

Profile Version - Type of transcoding used, based on your account configuration

  • Legacy - Legacy transcoding
  • Dynamic Delivery - Stores a small set of renditions and performs real-time packaging.
  • Context-Aware - Uses machine learning to optimize the number and quality of renditions needed.

Type - Type of ingest profile

  • Standard VOD - Default profile provided by Brightcove. This cannot be edited or deleted.
  • Custom VOD - Custom profile that you have created.
  • Standard Live - Default profile provided by Brightcove. This cannot be edited or deleted.
  • Custom Live - Custom profile that you have created.

Renditions - Number of renditions that may be created

Display - Toggle to hide/display the profile from the Upload page

Delete - Click the delete icon (trash can) next to the ingest profile, and then click Delete Profile.

New Profile

  • Dynamic Delivery - Create a new Dynamic Delivery profile
  • Live - Create a new Live profile
