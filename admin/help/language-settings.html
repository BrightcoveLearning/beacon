Language Settings

This is where you define the languages associated with your video metadata..

If you delete a language that is in use, all metadata associated with the language will be hidden. To restore the associated metadata, add the language again.

Primary Language

  • The Primary Language is initially displayed when you edit video properties in the Media module. Existing metadata will be associated with the Primary Language.
  • Click on the Primary Language displayed.
  • Start typing your language to see values.
  • Select a language.
  • Click Save.

Additional Languages

  • After a Primary Language is saved, additional metadata languages can be added. This allows you to add multi-lingual metadata in the Media module.
  • Click Add Language or click on a language in the list to edit.
  • In the Language input field, Start typing your language to see values.
  • Select a language.
  • Click Save Language.
