API Authentication

You will need access tokens to make calls to the Brightcove APIs, unless you've enabled basic authentication in Player Settings.

Here, you can define the scope of each access token by registering applications.

View application list

  • Register New Application - Click this button to create a new application associated with an access token.
  • Name - Click this link to view and edit the application details.
  • Date Issued - The date that the application was created.
  • Delete - Click this button to remove the application.

Register new or edit application

  • Name - The name of the application
  • Description - The description for the application.
  • Select Accounts for Authorization - Select one or more accounts for which you want to create access tokens.
  • Exposed Brightcove APIs - Select the API(s) and permission(s) to enable for this application. For security reasons, Brightcove recommends that you limit the scope of permissions.
  • Cancel - Cancel any changes made to this application.
  • Save - Save changes or create a new application.
  • Delete - Delete this application.
