How To: Add a User to an Account

In this topic you will learn how to add a user to a Brightcove Beacon account.

Steps

  1. Log in to Beacon Studio.
  2. In the left navigation header, click ADMIN and then User Management. Note that you must be an account administrator to access to the Administration page.
  3. Enter the Email, First Name and Last Name of the user.
  4. Assign a Role to the user:
    • Administrator - User has access to the ADMIN menu
    • Standard - User does not have access to the ADMIN menu
  5. Select the modules the user has access to. Turning off modules limits which sections of Studio the user has access to.
  6. Click Add User.

The user will receive an email message stating that they have been added to the account. The body of the page lists all users that have access to the account.
