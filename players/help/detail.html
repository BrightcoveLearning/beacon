Player Detail

View and set player details, including player information, styling, playback, advertising, and more.

Player Information

Use the player information properties to set the player name, sizing, aspect ratio, dimensions, update mode and player version.

  • Player ID - Player unique identifier
  • Name - Internal player name
  • Player Type - Determines if the player will display a single video or support a playlist
    • Single Video - The player will display a single video
    • Playlist - The player will support playlists
      Selecting Playlist will enable some additional properties in the Styling and Playback sections.
  • Short Description - An internal description for your player
  • Sizing - Set the player size
    • Responsive - The player size will adjust automatically based upon the browser size
    • Fixed - The player size is based upon the width and height parameters
  • Aspect Ratio - The ratio of the player's width to its height
    • 1:1, 3:2, 4:3, 16:9, 21:9 or 9:16 - When selected, enter the player width and the height is generated
    • Custom - Enter values for both player width and height
  • Player Dimensions - Control the width and height of the player; associated with aspect ratio
  • Update Mode - Control when the player is updated
    • Automatic
      • Default
      • Player is updated when Brightcove releases a new version of the player
    • Manual
      • Not recommended - you may miss bug fixes and enhancements
      • Lock a player to a version
      • Player is not automatically updated
  • Player Version - Related to Update Mode. For details, see the Setting the Player Update Mode and Version document
  • Published Player Version - The current version of this player
  • Localization - Localize the player interface elements to appear in various languages
    • Default is English
    • Include the language(s) you wish to use for player localization
  • Click Save to save your changes

Styling

Configure some basic styling options for a player.

To see playlist styling options, do the following:
  • In the left navigation, select Player Information
  • For the Player Type, select Playlist
  • Click Save
  • Playlist Type - Set how the playlist is displayed
    • Horizontal Playlist - Player will display a horizontal playlist
    • Vertical Playlist - Player will display a vertical playlist
    • Hidden Playlist - The playlist will be hidden
    Any changes made to a player may take up to 5 minutes to appear in published players.
  • Select Show title and description to display the video title and description on the player
  • Play Button Shape - Controls the shape of the play button:
    • Circle
    • Square
    • Rectangle
  • Play Button Position - Controls the location of the play button within the player:
    • Center
    • Top Left
    • Top Right
  • Color Theme - Controls the colors used in the player including the play/pause buttons, elapsed time, volume, captions, full screen and sharing icons.
  • Logo Overlay - Add an image on top of the player:
    • On - Upload an image and position it:
      • Bottom Right
      • Bottom Left
      • Top Right
    • Off - No image
  • Click Save to save your changes

Playback

Set video playback properties for a player.

To see playlist playback options, do the following:
  • In the left navigation, select Player Information
  • For the Player Type, select Playlist
  • Click Save
  • Auto-Play Video on Player Load - Start playing a video when the player is loaded
    • Auto-play may function differently depending on the browser
    • For details, see the Autoplay Considerations document
  • Play Muted - Allows you to mute video playback
  • Preload Video Before Playback - Controls whether the player will preload video content before the user initiates playback
    • Preload Video (fastest) - The entire video file is downloaded, even if never watched. This will increase bandwidth consumption, but videos play immediately
    • Preload Metadata - Default: only video metadata is preloaded
    • Do not preload - None of the video is preloaded
    • For details, see the preload details section
  • Enable DRM - enables the player to play DRM-protected content
  • Playlist properties - Additional properties when a Player Type of Playlist is selected in the Player Information section:
    • Randomize play order - Videos are played in a random order
    • Play video when selected - Play a video when selected from a playlist, even if the previous video was paused
    • Continuous play mode - Videos advance automatically and repeat indefinitely
    • Video Countdown - Enabled when Continuous play mode is checked
      • Up next card (auto advance) - An up next card is displayed in the player
      • Up next endscreen (countdown) - An end screen countdown is displayed in the player

Controls

Configure the player control bar.

  • Quality Selector - Allows the user to manually select a different rendition while the video is playing
    • Show Resolution (e.g., 720p) - Renditions will be displayed by lines of horizontal resolution
    • Show HD/SD - The player uses bitrate information to determine if a rendition is HD or SD
  • Playback Rates - Allows the user to manually select a different playback rate
  • Fullscreen - Enables the fullscreen control to resize the player to fill the entire display
  • Vertical Volume Slider - Displays the volume slider vertically
  • Thumbnail Seeking - Enables the display of video thumbnails over the progress bar
  • Skip to next video (playlist) - Enables the skip control to allow users to advance to the next video in the playlist

Social

Configure social sharing properties for a player.

  • Enable Sharing - When enabled, a share icon will appear; select the social media sites that the video can be shared to
  • Display Player Embed Code - When social sharing options are displayed, allows the user to copy the player embed code
  • Display Player Direct Link - When social sharing options are displayed, allows the user to copy a link to the page where the player is embedded

Advertising

Configure a player to to support client-side and/or server-side advertising.

When you enable both client-side and server-side ads, ad failover is automatically turned on. This means when Brightcove Player detects an ad blocker in the browser, the player requests server-side ads. Otherwise, it uses client-side ads.
  • Enable Client-Side (IMA) - Configure client-side advertising for a player
    • Ad Tag - Enter your ad tag URL; for testing, select Insert sample Ad Tag
    • Request Ads - Set when ads are requested
      • On load - Ads are requested immediately when the player loads (normally the best experience for DFP / VPAID)
      • On play - The first ad request is delayed until playback is initiated
      • On demand - All ad requests are initiated programmatically using the player.ima3.adrequest() method
    • VPAID Mode - VPAID Mode is used to enable VPAID 2 support in IMA ads
      • Enabled - Play VPAID ads in an iframe with a different domain
      • Insecure - Play VPAID ads in an iframe with the same domain
      • Disabled - VPAID ads throw an error
    • Timeout - The maximum amount of time to wait, in milliseconds, for an ad to initialize before playback
    • Hard timeouts - Prevents slow loading ads from appearing; unchecking this option could result in a slow loading ad interrupting video playback
    • Maximum Number of Redirects - Set the maximum number of redirects before subsequent redirects are denied, and the ad load is aborted; the number of redirects directly affects latency and user experience
    • Plugin Version - Select the IMA plugin version to use
    • Click Save
  • Enable Server-Side (SSAI) - Configure server-side advertising for a player
    The server-side advertising option will only appear if your account has been enabled for server-side ads.
    To get accurate ad impressions, change your settings in the Playback section to do not preload.
    • Select Configuration - Select an ad configuration to associate with this player
    • Enable ad information overlays - Enables the player to display overlays over your ads; this includes Learn More and ad count down overlays
    • Click Save

Content Restriction

Configure a player to restrict playback to specific domains, and allow/block viewing from specific countries.

Domain Restriction is available to all accounts. Geo Restriction is only available to Enterprise and Professional accounts.
  • Allowed Domains - Enter one or more domain URLs to restrict a player's availability to these domains
    • You must use complete URLs without wildcards
    • URLs must not end with a /
    • Only domains, not specific paths, are supported. For example:
      • Yes: https://www.brightcove.com
      • No: https://www.brightcove.com/documentation
    • Click Save
  • Geo Filtering - control which countries can, and cannot, use a player
    1. Select one:
      • Unrestricted
      • Allow viewing from the following countries
      • Prevent viewing from the following countries
    2. Enter the country names. As you type, the country list will be filtered.
    3. Click on a country and press enter to add it
    4. Click Save
