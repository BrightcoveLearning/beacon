Available functionality/information

This home page of the Layout module provides the following functionality/information:

Page listing

Each page created is listed in the page list. The following details the listing:

  • Click and Drag Handle: Icon indicating the page can be moved using click and drag. You can click anywhere on the page name to move it.
  • Page Type Indicator:: Indicates the page type, from the following:
    • : Home page
    • : System Page
    • : User Configurable Page
  • Page Title: Title of the page. The gray name below the title is the system name of the page.
  • Page Availability Indicator: If displayed, indicates availability settings are currently preventing the page from being displayed in the apps.
  • Duplicate Button: Click to make a duplicate of the page. The duplicate name will be in the format Page Name (copy).
  • Page Settings Dialog Button: Click to open the page settings dialog. Here you can change the page title, availability settings and archive the page.

Create page

Click Create Page to start the page creation process.

Delete or restore archived pages

Click Archived Pages to display a list of all archived pages from which you can choose to delete or restore the archived page.
