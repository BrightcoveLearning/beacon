Pages

Select a page type.

  • Customize with your content - Select this option to be able to add playlists or channels to a page. Pages will also have multiple layout options.
  • Choose from a template - Select a pre-configured page layout, typically used for Favorites, Recently Watched, EPG and Live TV type pages. Layout for these types of pages is mostly fixed.
