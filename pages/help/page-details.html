Page Details

Click the gear icon () to edit the page details.

Click Add Playlist to select the playlist(s) that will be displayed on this page. If playlists have been selected, they will be displayed.

The Availability settings can be used to restrict page availability based upon:

  • Date
  • Device Type
  • Location
