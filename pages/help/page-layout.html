Page Layout

Select a page layout. The layout will control the look and feel of the page and determines the number of playlists that can be added to a page.

  • Tabs
  • Grid
  • Hero (Autoplay Channel)
  • Hero (Image)

When a page layout is selected, the page preview will update to reflect the current selection.

After making your selection, click the Select button that is positioned under the container types.


You can also click the gear icon beside the page name to edit the page title and availability.
