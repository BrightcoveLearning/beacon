Page Layout

Select a page layout. The layout will control will control the look and feel of the page and will control the amount of content (playlists) that can be added to a page.

  • Playlist Grid
  • Hero (Autoplay Channel)
  • Hero (Featured Playlist)
  • Hero (Image)
  • Carousels

When a page layout is selected, the page preview will update to reflect the current selection.

Click Create to create the page.
