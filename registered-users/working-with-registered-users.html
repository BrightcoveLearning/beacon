Introduction

Registered users are users who have signed in or authenticated with your Brightcove Beacon app. If one of your registered users has a question or a problem with their account, you can quickly look them up using a smart search, and verify their details. User information can be found in the following tab sections:

  • Overview
  • Viewing history
  • Transactions
  • Active sessions
  • Profiles

Quick actions allow you to perform a global action from any of the tab screens.

Overview

The Overview tab displays general information about the selected user, including the plan names they have signed up for, and their parental control setting.

Viewing history

Viewing history displays a list of movies that this registered user has watched. History is captured and saved for 1 year.

Actions

  • Use the Show filter dropdown to limit display results

Columns

  • Date - The date that a movie was viewed; click this header to control the results sort order (descending or ascending)
  • Title - The name of the movie viewed
  • Profile - The user profile

Transactions

The Transactions tab shows a list of plan purchases and changes for a registered user.

Actions

  • Use the Show filter dropdown to limit display results

Columns

  • Store name - The app store name
  • Plan name - The subscription plan name
  • Status - The current subscription status; values: In Trial, Pending or Active
  • Start - The start date for the associated subscription
  • Trial ends - The end date for the user's trial period
  • Ends - The end date for the associated subscription
  • Canceled - The date that the subscription was canceled

Active sessions

View a list of active sessions associated with a registered user.

Actions

  • Click End all sessions to terminate all currently active user sessions

Columns

  • Store name - The app store name
  • Plan name - The subscription plan name
  • Status - The current subscription status; values: In Trial, Pending or Active
  • Start - The start date for the associated subscription
  • Trial ends - The end date for the user's trial period
  • Ends - The end date for the associated subscription
  • Canceled - The date that the subscription was canceled

Profiles

Thumbnail seeking allows users to drag the playhead along the timeline and view thumbnails as a preview of the associated content. This gives users the ability to quickly navigate a video file and find the content that they are interested in.
