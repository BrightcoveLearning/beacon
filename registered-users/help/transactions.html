User Transactions

Registered user transactions include plan purchases and changes.

Transactions

  • E-mail - Email address for the user
  • Name - The display name for the user
  • Last Login - The date that this user was last logged into your app
  • Plan - The type of monetization plan defined for this user; click a plan name to see details
  • Parental Control - Shows whether parental controls are enabled for this user
Delete and forget
WARNING: Deleting a viewer will delete all related data and permissions. This CANNOT be undone!
  1. In the left navigation, click Delete and forget.
  2. In the Delete and forget dialog, enter the word Delete.
  3. Click Delete and forget.
