Registered User Profiles

Here are the profiles associated with a registered user.

Top navigation

  • Quick actions
    • Delete and forget
      1. Select the transactions that you want to delete.
      2. Click Delete and forget to remove the selected transactions from the user log.

Profile details

  • Profile - The display name for the profile.
  • Created - The creation date for the profile.
  • Remove Profile - Click the delete icon associated with a profile to delete it.
