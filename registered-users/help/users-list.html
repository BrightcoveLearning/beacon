Registered Users List

Registered users have signed in or authenticated with your Brightcove Beacon app.

Search for a user

  1. Start typing either the email address or the name of the user you want to search for. You should see a list of users that match.
  2. Select a user from the search results list.
  3. The details page will open for the selected user.
  4. When you return to the list page, the viewed user will be added to the list.

View the users list

  • E-mail - Email address for the user.
  • Name - The display name for the user.

Edit a user

  1. Click on an E-mail link to view the user details.
  2. Click the Back to Users panel to return.
