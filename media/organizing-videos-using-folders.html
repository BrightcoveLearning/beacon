How To: Organize Videos Using Folders

In this topic you will learn how to add videos to folders.

Steps

  1. Open the Media page.
  2. In the side navigation panel, click the Create Folder link.
  3. Enter a name for the folder and then press enter.
  4. To add videos to a folder, select one or more videos.
  5. Click More > Move to Folder.
  6. Click on a folder and then click Move.

All folders will appear in the left navigation. The number after the folder name is the number of videos in the folder.

To remove videos from a folder, click on a folder, select one or more videos and then click Remove Videos > Remove Videos from Folder.
