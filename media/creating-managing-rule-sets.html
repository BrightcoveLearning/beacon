Creating and Managing Rule Sets

In this topic you will learn how to create and manage rule sets.

The availability settings for series and seasons can be configured so that access is based upon a start/end date, locations and/or devices. One or more rule sets can be created to allow you to configure when/where an asset will be available. To create and manage rule sets, edit a series or season and click the Availability tab.

All series must have at least one rule set. Rule sets are created to control access to a series based upon:

  • Start/end dates
  • Locations
  • Devices

Configuring schedule availability

Configure Schedule availability to configure the start and optional end date that a media asset will be available. By default, media assets are always available.

  1. Click Add New Rule Set to create a new rule set or hover over an existing rule set and click the edit icon ().
  2. Click the calendar icon () and pick a Start Date.
  3. Click the clock icon () and pick a start time.
  4. (Optional) Check the End Date checkbox and select an End Date and time.
  5. Click Done when finished.

Configuring device availability

Configure Device availability to configure the devices that a media asset can be viewed on. By default, media assets are available on all devices.

  1. Click Add New Rule Set to create a new rule set or hover over an existing rule set and click the edit icon ().
  2. Click on Devices to expand the device settings.
  3. Select the devices the media asset can be viewed on.
  4. Click Done when finished.

Configuring location availability

Configure Location availability to configure the locations that a media asset can be viewed from. By default, media assets are available in all locations.

  1. Click Add New Rule Set to create a new rule set or hover over an existing rule set and click the edit icon ().
  2. Click on Location to expand the location settings.
  3. Enter the Permitted Locations and Denied Locations for the media asset. Note that the locations that appear are based upon your account settings.
  4. Click Done when finished.
