Configuring Series Availability

In this topic you will learn how to configure series availability.

Series availability details can be used to restrict access to a series based upon date, locations and devices. To view and edit the availability details for a series, open the Media page and click on a series name link. The series overview page will open. Click the Availability tab.

The availability settings are controlled using rule sets. Rule sets are created to control access to a series based upon:

  • Start/end dates
  • Location
  • Device

Configuring schedule availability

Configure Schedule availability to configure the start and optional end date that a series will be available. By default, series are always available.

  1. Open the Media page.
  2. Click on a series name link.
  3. Click the Availability tab.
  4. Click Add New Rule Set to create a new rule set or hover over an existing rule set and click the edit icon ().
  5. Click on Schedule to expand the schedule settings.
  6. Click the calendar icon () and pick a Start Date.
  7. Click the clock icon () and pick a start time.
  8. (Optional) Check the End Date checkbox and select an End Date and time.
  9. Click Done when finished.

Configuring device availability

Configure Device availability to configure the devices that a series can be viewed on. By default, series are available on all devices.

  1. Open the Media page.
  2. Click on a series name link.
  3. Click the Availability tab.
  4. Click Add New Rule Set to create a new rule set or hover over an existing rule set and click the edit icon ().
  5. Click on Devices to expand the device settings.
  6. Select the devices the series can be viewed on.
  7. Click Done when finished.

Configuring location availability

Configure Location availability to configure the locations that a series can be viewed from. By default, series are available in all locations.

  1. Open the Media page.
  2. Click on a series name link.
  3. Click the Availability tab.
  4. Click Add New Rule Set to create a new rule set or hover over an existing rule set and click the edit icon ().
  5. Click on Location to expand the location settings.
  6. Enter the Permitted Locations and Denied Locations for the series. Note that the countries that appear are based upon your account settings.
  7. Click Done when finished.
