Configuring Season Availability

In this topic you will learn how to configure season availability.

Season availability details can be used to restrict access to a season based upon date, locations and devices. To view and edit the availability details for a season, open the Media module and click on a series name link. The series overview page will open. On the right side of the page, click on a season. The season Overview page will open. Click the Availability tab.

Seasons can inherit the availability settings from the season or you can configure customized availability settings.

Inheriting series availability

To have a season inherit it's availability settings from a series, select the Inherit option. By default, seasons will inherit their availability settings from the season. The calendar, location and device availability settings for the series will be applied to the season and will be displayed in the season availability settings.

Customizing season availability

To customize the season availability settings, select the Customize option. etails can be used to restrict access to a season based upon date, locations and devices. To view and edit the availability details for a series, open the Media module and click on a series name link.

Configuring device availability

Configure Device availability to configure the devices that a series can be viewed on. By default, series are available on all devices.

  1. Open the Media module.
  2. Click on a series name link.
  3. Click the Availability tab.
  4. Click Add New Rule Set to create a new rule set or hover over an existing rule set and click the edit icon ().
  5. Click on Devices to expand the device settings.
  6. Select the devices the series can be viewed on.
  7. Click Done when finished.

Configuring location availability

Configure Location availability to configure the locations that a series can be viewed from. By default, series are available in all locations.

  1. Open the Media module.
  2. Click on a series name link.
  3. Click the Availability tab.
  4. Click Add New Rule Set to create a new rule set or hover over an existing rule set and click the edit icon ().
  5. Click on Location to expand the location settings.
  6. Enter the Permitted Locations and Denied Locations for the series. Note that the countries that appear are based upon your account settings.
  7. Click Done when finished.
