Publishing a Single Video to the Web

In this topic you will learn how to publish a single video to the web.

Steps

  1. Open the Media module.
  2. Select a video to publish by checking the box next to the video thumbnail.
  3. Click Publish and Embed > Web Player.
  4. In the Select Player dropdown list, select a player to use.
    Note: The Brightcove Default Player is created by default in all Video Cloud accounts.
  5. Click the URL below the preview player to open it in a new browser tab. Confirm that the video appears and can be played.
  6. Copy the embed code and paste it into your webpage.
