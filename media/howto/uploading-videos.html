How To: Upload Videos

Steps

  1. In Beacon Studio, click All videos in the panel navigation.
  2. Click Upload to open the Upload panel.
  3. Use the file picker to select one or more files to upload. You can also close the file picker and drag and drop files from the file system to the Upload panel.
  4. The files will begin processing. Remain in the Media module (and don't refresh the page) until all videos have a status of Processing Complete.
  5. Click X to close the panel.
