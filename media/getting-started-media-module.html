Getting Started with the Media Module

This topic provides an overview of the Media module which is used to manage series, videos and playlists in a Brightcove Beacon account.

Accessing the Media module

To access the Media module, login to Beacon Studio and click Media in the primary navigation.

The body of the page will display a list of all series in the account.

The side navigation panel is organized into three main sections:

  • Programs - Used to view and create series. See Managing Series for information.
  • Videos - Used to view videos and create folders. See Managing Videos for information.
  • Playlists - Used to view and create playlists. See Introduction to Playlists for information.

The side navigation panel is displayed by default.

The side navigation button can be used to hide and show the side navigation panel. The size of the side navigation panel can be changed by clicking on the black border and dragging.

Keyboard shortcuts

Pressing the ? key in the Media module will open a list of the shortcut keys that are available.
