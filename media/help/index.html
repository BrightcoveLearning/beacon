Media Home

The Media page is used to upload and manage videos and to create series and seasons.

Upload Videos
  1. Click Browse for Videos and browse and select one or more files to upload or Drag and Drop files from the file system to the Media page.
  2. The files will begin processing. Remain on the Media page (and don't refresh the page) until all videos have a status of Processing Complete.
  3. Click View and edit video details to view the video and edit the video details in the Media page.
Search Videos
To search for a series, season, episode or movie, enter search terms and press enter. The following video fields will be searched:
  • Video id
  • Name
  • Short description
  • Long description
  • Tags
  • Reference id
  • Custom fields
Edit Video Details for Multiple Videos
Quick Edit is used to edit the video details for multiple videos.
  1. Select the videos to edit by clicking the checkbox next to the video name(s).
  2. Click Quick Edit.
  3. Use the Quick Edit panel to edit the video details as needed.
  4. Click Save to save the changes.
Move a Video to a Folder
Folders can be used to organize videos.
  1. In the side navigation panel, click the Create Folder link.
  2. Enter a name for the folder and then press enter.
  3. To add videos to a folder, select one or more videos.
  4. Click More > Move to Folder.
  5. Click on a folder and then click Move.
Create a Manual Playlist
  1. In the side navigation panel, click the Create Playlist link.
  2. Set the Playlist type to Manual.
  3. Enter a Playlist Name.
  4. Check Show Playlist in Sidebar to have the playlist appear in the sidebar.
  5. Click Save.
  6. To add videos to the playlist, click All Videos in the side navigation panel.
  7. Click the checkbox next to videos you want to add to the playlist.
  8. Click More > Add To Playlist.
  9. Select a playlist and then click Add.
