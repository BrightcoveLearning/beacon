Upload

The Upload page is used to replace the source file for an existing video.

  1. Select an Ingest Profile.
  2. Drag and Drop files from the file system to the Upload module OR Click and browse and select one or more files to upload OR click and enter the URL to a remote video file.
  3. The file will begin processing. Remain on the Upload module page (and don't refresh the page) until the video has a status of Processing Complete.
