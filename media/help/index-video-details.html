Video Details

The video details are organized into sections. Click or double-click in a section. Some sections allow the details to be edited directly.

Click Save to save the changes. Some sections do not have a Save button and will autosave when tabbing out of the field.

Video Information

Contains basic information about a video.

  • Name - Max 255 characters.
  • ID - System assigned video ID (unique across all Brightcove accounts).
  • Short description - Max 250 characters.
  • Reference ID - A unique ID assigned by a publisher; max 150 characters
  • Video Projection - Indicates how the video was recorded.
  • Enable offline playback - When checked, enables the video to be saved locally and viewed offline
  • Related Link URL - A standard web URL beginning with http:// or https:// limited to 255 characters.
  • Related Link Text - Related link text limited to 255 characters.
  • Long Description - Max 5000 characters.
  • Advertising
    • Free - Ads will not be shown when the video plays.
    • Ad Supported - Ads may be shown when video plays.
  • Ad Keys - Passed as part of the ad server URL.
  • Sharing - Indicates if the video has been shared to another account.

  • All fields support type-ahead search. As you type, previous matching entries will appear and can be clicked on to add them to the selection.
Custom Fields
  • Used to view and edit the custom field for the video
  • To edit custom field values, click the Edit button or double-click in the section
  • An asterisk (*) next to the field name indicates that the field is required.

Video Type Information
  • Select a Video Type for the video
  • The Video Type determines additional video details that are available to be configured
  • The following Video Types are available:
    • Movie - Indicates the video is a movie. When a new video is uploaded, the Video Type will be set to Movie.
    • Episode - Indicates the video is an episode as part of a season. The Video Information panel will display the episode number and the season the video is assigned to.
    • Trailer - Indicates the video is used as a trailer. Videos that are added as a trailer for a movie or episode will automatically have their Video Type changed to Trailer.
Categorization

The Categorization section can be used to configure:

  • Tags - One or more tags, separated by commas. You can use tags to add videos to smart playlists. Max 128 characters.
  • Manual Playlists - Playlists that this video belongs to. For more information, see Creating and Managing Playlists .
  • Folder - Folder the video is assigned to. See Organizing Videos Using Folders for more information.
Availability
  • Used to control the dates and times that a video can be viewed
  • Geo filtering can be used to control which countries have the ability to view the video
