The Create Live Event page is where you will create your live event providing the required information.

To create a new event:

  1. Enter a Event Name.
  2. Enter a Short Description of the event (max 250 characters).
  3. Enter Tags optionally to associate with the event. Those will be assigned to the video if the video is saved as a video asset.
  4. Enter a Reference ID as an optional identifier for internal use.
